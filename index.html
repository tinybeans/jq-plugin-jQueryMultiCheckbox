jQuery Multi-checkbox のサンプル

サンプル1

jQuery(function($){
    $('.multicheck').multicheckbox();
});

<p><input class="multicheck" type="text" title="バッジョ,ジーコ,クライフ,ロマーリオ" value="" /></p>

サンプル2（showオプションを設定：テキストフィールドを表示して確認）

jQuery(function($){
    $('.multicheck2').multicheckbox({show:'show',add: true,});
});

<p><input class="multicheck2" type="text" title="バッジョ,ジーコ,クライフ,ロマーリオ" value="バッジョ,クライフ,ピルロ" /></p>
「バッジョ,クライフ」はチェックされていた項目、「ピルロ」はユーザーが追加した項目と仮定する

サンプル3（labelオプションを設定）

jQuery(function($){
    $('.multicheck3').multicheckbox({label:'ブラジル,ポルトガル,スペイン,オランダ'});
});

<p><input class="multicheck3" type="text" value="" /></p>

サンプル4（show,labelオプションを設定）

jQuery(function($){
    $('.multicheck4').multicheckbox({show:'show',label:'ブラジル,ポルトガル,スペイン,オランダ,イン',sort:'ascend'});
});

<p><input class="multicheck4" type="text" value="" /></p>

サンプル5（labelオプションにオブジェクトを設定、insert:'after'を設定）

jQuery(function($){
    $('.multicheck5').multicheckbox({show:'show',insert:'after',add: true,label:{
            "Brazil":"ブラジル",
            "Portugal":"ポルトガル",
            "gal":"ガル",
            "Spain":"スペイン",
            "Holland":"オランダ"
        },
        sort: 'ascend'
    });
});

<p><input class="multicheck5" type="text" value="Brazil,Spain,Japan" /></p>
「Brazil,Spain」はチェックされていた項目、「Japan」はユーザーが追加した項目と仮定する
