※すべて「show: 'show'」を設定

jQuery Multi-checkbox のサンプル

サンプル1（デフォルト）

$('.multicheck').multicheckbox();

<input class="multicheck" type="text" title="バッジョ,ジーコ,クライフ,ロマーリオ" value="" />

サンプル1-2（add: true）

$('.multicheck').multicheckbox({add: true});

<input class="multicheck" type="text" title="" value="" />

サンプル2（add: true）

$('.multicheck2').multicheckbox({show:'show',add: true});

<input class="multicheck2" type="text" title="バッジョ,ジーコ,クライフ,ロマーリオ" value="バッジョ, クライフ, ピルロ" />
「バッジョ,クライフ」はチェックされていた項目、「ピルロ」はユーザーが追加した項目と仮定する

サンプル3（label: 文字列）

$('.multicheck3').multicheckbox({show:'show',label:'ブラジル,ポルトガル,スペイン,オランダ'});

<input class="multicheck3" type="text" value="" />

サンプル4（label: 文字列,sort:'ascend',add: true）

$('.multicheck4').multicheckbox({show:'show',label:'ブラジル,ポルトガル,スペイン,オランダ,イン',sort:'ascend',add: true});

<input class="multicheck4" type="text" value="" />

サンプル5（label: オブジェクト,show:'show',insert:'after',add: true）

$('.multicheck5').multicheckbox({
    show:'show',
    insert:'after',
    add: true,
    label:{
        "Brazil":"ブラジル",
        "Portugal":"ポルトガル",
        "Gal":"ガル",
        "Spain":"スペイン",
        "Holland":"オランダ"
    }
});

<input class="multicheck5" type="text" value="Brazil,Spain,Japan" />
「Brazil,Spain」はチェックされていた項目、「Japan」はユーザーが追加した項目と仮定する

サンプル6（label: オブジェクト,show:'show',insert:'after',add: true,tags:'tags',sort: 'ascend'）

$('.multicheck6').multicheckbox({
    show:'show',
    insert:'after',
    add: true,
    tags:'tags',
    label:{
        "Brazil":"ブラジル",
        "Portugal":"ポルトガル",
        "Gal":"ガル",
        "Spain":"スペイン",
        "Holland":"オランダ"
    },
    sort: 'ascend'
});


<input class="multicheck6" type="text" value="Brazil,Spain,Japan" />
「Brazil,Spain」はチェックされていた項目、「Japan」はユーザーが追加した項目と仮定する
